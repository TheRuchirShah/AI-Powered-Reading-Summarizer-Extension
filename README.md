# AI-Powered-Reading-Summarizer-Extension

A **browser extension** that cleans up article pages (like Reader Mode) and generates a **3-point AI summary**.


## Features

- Removes ads, sidebars, headers, and footers for a clean reading experience.
- Generates AI-powered summaries using OpenAI/Gemini.
- Displays the summary in a popup panel.
- Cross-browser support (Chrome & Firefox).
